# 创业模式探讨与思考

---

## 一、赚钱的底层模式

从"钱是怎么赚到的"这个最底层逻辑来拆，只有五种基本模式：

| 模式 | 本质 | 典型案例 | 杠杆倍数 |
|---|---|---|---|
| **卖时间** | 用时间和技能换钱 | 咨询、自由职业、律所 | 1x（一份时间一份钱） |
| **套利** | 利用信息差/价格差/地域差 | 跨境电商、流量套利、贸易 | 不稳定，窗口期短 |
| **创造** | 做出原本不存在的东西 | 产品、内容、技术、品牌 | Nx（做一次卖 N 份） |
| **收租** | 拥有稀缺资源，重复收费 | SaaS、专利、IP 授权、房产 | 持续被动收入 |
| **撮合** | 连接供给和需求 | 平台、中介、广告、保险、VC | 指数（网络效应） |

### 几个容易混淆的模式归类

- **卖铲子**（如 NVIDIA 卖 GPU、AWS 卖云服务）→ 撮合的特殊形态，把工具供给和淘金需求连起来
- **承担风险**（如保险、风投）→ 也是撮合，把资本和风险连起来
- **聚合**（如社群、集采、拼团）→ 不是独立的赚钱模式，而是走向撮合的必经能力。聚了人之后，最终还是靠收租（会员费）、撮合（广告）、创造（卖课卖货）或套利（集采议价）来变现

### 跃迁路径

每一次跃迁，杠杆都放大一级：

```
卖时间 → 套利/创造 → 收租 → （通过聚合能力）→ 撮合
```

很多创始人的痛苦是卡在了前面的模式里出不来。比如做咨询的人很难跳到做产品，因为咨询来钱快、产品投入大，短期看咨询更"理性"——但长期看，你永远在卖时间。

## 二、创业的三种路径模式

### 模式一：探索式（Emergent）

- 从一个具体的、可盈利的切入点开始
- 在"做"的过程中积累认知壁垒，发现更大的机会
- 典型路径：咨询 → 产品 → 平台
- 代表案例：Basecamp（从接外包到做项目管理工具）、Shopify（从卖滑雪板到做电商平台）
- 通常对应自负盈亏的做法

### 模式二：愿景式（Visionary）

- 先定义终局形态，倒推里程碑和 roadmap
- 需要在市场验证前大量投入，所以通常需要融资
- 代表案例：SpaceX、OpenAI、Tesla

### 模式三：杠杆式（Leverage）

- 先积累某种可复用的杠杆（个人品牌、流量、技术能力、用户关系），然后用这个杠杆去"撬"商业机会
- 代表案例：网红/KOL 转品牌、开源项目转商业化
- 在 AI 时代特别值得关注，因为 AI 本身就是一种超级杠杆

### 如何选择？

关键问题不是选哪种模式，而是：**你目前手里最强的牌是什么？**

- 手里有现金流能力 → 模式一更安全
- 手里有稀缺洞察且时间窗口紧 → 模式二可能必须选
- 手里有杠杆（影响力/技术/社区） → 模式三值得考虑

## 三、创业的两个阶段：搜索与执行

创业不是选一种模式走到底，而是两个阶段的切换：

| | 搜索期 | 执行期 |
|---|---|---|
| 核心任务 | 找到 PMF（Product-Market Fit） | 规模化、建壁垒 |
| 思维模式 | 发散、试错、快速迭代 | 收敛、聚焦、长期投入 |
| 资源分配 | 小成本多方向试探 | 集中资源打穿一个点 |
| 危险 | 找不到方向 / 过早放弃 | 方向错了还在死磕 |

### 探索式创业的核心风险：机会主义漂移

探索式创业最大的危险不是"找不到机会"，而是**机会太多**。不断发现新机会会导致：

- **团队疲于奔命**：每个新机会都觉得值得试，资源不断被拉扯
- **没有复利积累**：做了 A 又做 B，A 的能力和资源在 B 上用不上，永远在重启
- **变成"项目公司"而非"产品公司"**：靠不断接新活存活，本质上和外包没区别
- **团队丧失信念感**：大家不知道公司到底要去哪，只知道"老板又有新想法了"

这是很多中小公司卡在几百万到几千万营收就上不去的核心原因。

### 关键动作：收敛

探索和愿景并不矛盾，真正的区别在于——**你有没有在探索过程中"收敛"的纪律。**

好的探索：

```
探索 → 发现机会 → 验证 → 【收敛】→ 全力投入 → 再探索边界
```

坏的探索：

```
探索 → 发现机会A → 做A → 发现机会B → 也做B → 发现C → 也做C → 精力崩溃
```

### 何时该收敛？四个信号

1. **有一个方向的增长是"拉着你走"的** —— 不是你在推，而是客户/用户在拉你
2. **你能用一句话说清楚你在做什么** —— 如果解释公司要五分钟，说明还没收敛
3. **团队对一个方向有体感共识** —— 不是老板拍脑袋，而是一线也觉得"就是这个"
4. **砍掉其他业务你不会心疼** —— 如果每条线都舍不得，说明还没找到那个真正有势能的点

### 何时该放弃？四个信号

1. **你自己已经不相信了**，但还在靠惯性和责任感维持
2. **核心假设被证伪了**，不是执行问题，而是方向本身不成立
3. **机会成本太高**，你留在这里意味着错过了一个明显更好的机会
4. **身体或心理发出了严重警告**

放弃不是失败。**在错误的方向上坚持，才是最大的浪费。**

## 四、热爱与市场的矛盾

### 热爱的价值

热爱带来的确定性是极其稀缺的资源，它解决了创业里最难的问题——**坚持的理由**。但热爱要转化为创业，中间需要一个翻译层——这个热爱对应的市场是否存在。

- **热爱 + 市场存在** = 最理想的状态（Patagonia、任天堂宫本茂）
- **热爱 + 市场不存在** = 艺术家，不是创业者（不是贬义，只是不同的路）
- **不热爱 + 市场存在** = 能赚钱，但会在某个阶段失去动力

### 撕裂时刻：被市场拉向不想去的方向

很多创始人都经历过：

- Slack 的创始人本来想做游戏，游戏失败了，内部通讯工具反而火了
- Flickr 本来是游戏公司的副产品
- YouTube 最初是约会网站

面临的选择：

> **A：跟着市场走，放弃初心，做那个"对的但不爱的"事情**
> **B：坚持初心，放弃被验证的市场机会**

### 核心解法：重新定义热爱的抽象层级

- **热爱定义得很具体**（我要做某个特定的产品）→ 跟市场冲突的概率很高
- **热爱定义在更上一层**（我想解决某类人的某类痛苦 / 我想用技术改变某个领域的效率）→ 市场的反馈不是"背叛"，而是在帮你找到实现热爱的更有效路径

例如：

- "我热爱做 AI 绘画工具" → 具体，容易被市场否定
- "我热爱让普通人拥有创造力" → 上位概念，AI 绘画只是一种实现方式，市场可能告诉你更好的方式是 AI 视频、AI 设计模板、甚至 AI PPT

**重新定义热爱的抽象层级，是创始人最重要的心智操作之一。**

### 也要对自己诚实

如果内心清楚地知道"我就是想做这一件具体的事"，也完全可以。只是要接受：

1. 可能需要更长的时间才能找到市场
2. 可能永远找不到足够大的市场
3. 做的可能更接近"生意"或"工作室"，而不是"可规模化的公司"

**这没有高低之分。** 一个人做自己热爱的小而美的事业，活得充实又自由，不比做一个上市公司差。

## 五、创始人要素

### 时机（Timing）

Bill Gross 分析了 200 多家创业公司后发现，成功的第一因素不是团队、不是商业模式，而是**时机**。太早了市场教育成本极高，太晚了红利已经被吃完。

- 太早的例子：WebVan（1999 年做生鲜电商，概念对了但基础设施没准备好）
- 刚好的例子：Zoom（视频会议技术早就有了，但 2020 年疫情把时机推到了完美窗口）

时机无法完全预判，但可以感知信号：**当你发现解释成本突然降低了**（不用花力气说服别人这事有价值），说明时机到了。

### 认知带宽

创业过程中，创始人最稀缺的资源不是钱，不是人，而是**认知带宽**。你能同时思考多少层面的问题、能多快地学习新领域、能不能在信息不完整时做出大致正确的判断——这些决定了公司能走多远。

具体表现：
- 能不能从执行层抬起头来看战略？
- 能不能在焦虑和压力下保持判断力？
- 能不能听进去跟自己直觉相反的反馈？

保护认知带宽的方式：减少不必要的决策（把能标准化的事情标准化）、有意识地留出"不做事"的思考时间、找到能补足自己短板的人。

### 合伙人

独自创业最大的风险不是能力不够，而是没有人在你动摇的时候把你拉回来，也没有人在你盲目自信时给你泼冷水。

选合伙人的核心原则：
- **能力互补**比志趣相投更重要——两个都爱做产品的人在一起，谁去盯销售？
- **在压力下还能沟通**比平时关系好更重要——顺风时谁都处得来，逆风时才见真章
- **对"什么是成功"的定义一致** —— 一个人想做十亿美金公司，一个人想做小而美赚钱的生意，迟早会分裂

## 六、AI 时代的变量

### 对创业模式的重塑

- **团队规模假设被打破**：以前需要 20 人做的事，现在 3-5 人就可以，更多人可以选择自负盈亏，因为启动成本大幅降低
- **技术壁垒在消融**：AI 让技术不再是稀缺要素，分发能力、行业认知、用户信任变成了更重要的壁垒
- **迭代速度成为核心优势**：谁能更快地用 AI 把想法变成产品、投入市场、拿到反馈、再迭代，谁就赢了
- **个体杠杆空前放大**：一个有判断力的人 + AI 工具链，能产出过去一个小团队的成果

### 对赚钱模式的影响

| 模式 | AI 时代趋势 |
|---|---|
| **卖时间** | AI 放大个体产出，但仍受时间约束 |
| **套利** | AI 加速信息透明，套利窗口越来越短 |
| **创造** | 创造效率被 AI 放大，但方向和品味仍稀缺 |
| **收租** | 好的租值资产更值钱，差的被 AI 替代品冲击 |
| **撮合** | 纯撮合门槛降低、竞争更激烈，但有网络效应的平台护城河依然深 |
